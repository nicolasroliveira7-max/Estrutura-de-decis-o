# Estrutura-de-decis-o
Uma estrutura de decisão (ou estrutura condicional) é um bloco de lógica na programação que permite ao sistema desviar o fluxo de execução. Ela funciona como uma pergunta: se uma condição for verdadeira, o código executa uma ação; se for falsa, executa outra.

Entenda os principais tipos utilizados na lógica de programação:
-Simples (SE / if): Executa um bloco de código apenas se a condição testada for verdadeira.
-Composta (SE-SENÃO / if-else): Oferece dois caminhos exclusivos. Um bloco roda se a condição for verdadeira e outro se for falsa.
-Encadeada (ou Aninhada): Utiliza várias decisões em cascata (uma dentro da outra) para verificar múltiplos cenários ou faixas de valores (ex: classificar notas de alunos em A, B, C).
-Múltipla Escolha (switch-case): Ideal para comparar uma mesma variável com vários valores ou opções fixas (ex: criar um menu de opções onde o usuário digita 1, 2 ou 3).
